== Description ==

Easy Filter is a WordPress CMS plugin To give more custom filter option in list pages(posts/pages/any custom post types).
The Filter Option is customizable from settings page.


== Installation ==

Follow this steps to install **Easy Filter**

1. Download the plugin into the **/wp-content/plugins/** folder
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure Your Filter Option through The 'Easy Filter' menu under 'Settings' menu in your WP admin panel

== Frequently Asked Questions ==

= How To configure the custom filter options?  =

All the settings option can be accessed through The 'Easy Filter' menu under 'Settings' menu in your WP admin panel

= I have Install The Plugin And activated! Now What? =

Configure Your Filter Option through The 'Easy Filter' menu under 'Settings' menu in your WP admin panel. By Default the filter option are same as wordpress default behaviour

= When Date Range Filter Option Selected, The Date range field showing properly but The Calender Picker Is not showing properly  =

If you mean to say the style of calender picker, You may be using the plugin locally without any Internet access!! The jquery UI style assets are loaded from google server. This should work fine online!!

= I have Selected The Taxonomy Filter For a 'Post Type' But 'Select Taxonomy Input Style' Section are not showing  =

May Be There is no taxonomy found for the **Post Type**
